## Задание 2. Теги

### GitHub:

Оба тега (легковесный v0.0 и аннотированный v0.1) отображаются в списке тегов.
Для обоих тегов отображается дата создания (Yesterday).
Для обоих тегов отображается SHA коммита, на который они указывают.
Для обоих тегов доступны ссылки для скачивания архива (zip и tar.gz).
Аннотированный тег v0.1 не имеет расширенного описания или сообщения в этом представлении.

### GitLab:

Оба тега (легковесный v0.0 и аннотированный v0.1) отображаются в списке тегов.
Для обоих тегов отображается SHA коммита, на который они указывают.
Для аннотированного тега v0.1 отображается сообщение, добавленное при создании тега (“Release v0.1: Initial release with basic features”).

### Вывод:

В целом, обе платформы отображают основную информацию о тегах (имя, SHA коммита, дату). Однако, GitLab более явно показывает сообщение, связанное с аннотированным тегом, что делает его более информативным в представлении списка тегов. В GitHub для просмотра сообщения аннотированного тега может потребоваться перейти на страницу “Releases” или просмотреть информацию о теге.
